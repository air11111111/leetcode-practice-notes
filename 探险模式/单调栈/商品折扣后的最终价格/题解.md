# LeetCode：商品折扣后的最终价格

## 题目信息

- 题目链接：https://leetcode.cn/problems/final-prices-with-a-special-discount-in-a-shop/description/?envType=problem-list-v2&envId=shujujiegouyusuanfa-xianxingqiantan-dandiaozhan
- 难度：简单
- 核心知识点：栈、数组、单调栈

## 题目概述

给定商品价格数组，对于每个商品，需要在它右侧找到第一个价格小于等于它的商品作为折扣，并返回每个商品折扣后的最终价格。

## 解题思路

代码使用双重循环直接模拟。外层枚举每个商品 i，内层从 i+1 开始向右查找第一个 prices[j] <= prices[i] 的商品；找到后从当前价格中减去该折扣并立即停止查找。如果右侧没有满足条件的价格，当前商品价格保持不变。这种写法没有维护额外栈结构，空间更省，但需要为每个位置向右查找，时间复杂度会更高。

## 解题步骤

1. 从左到右枚举每个商品价格。
2. 对每个商品，从它右侧第一个位置开始查找可用折扣。
3. 遇到第一个不大于当前价格的商品时，减去该价格并停止内层循环。
4. 遍历结束后返回修改后的价格数组。

## 我的 LeetCode 代码

```cpp
class Solution {
public:
    vector<int> finalPrices(vector<int>& prices) {
        for(int i=0;i<prices.size();i++){
            for(int j=i+1;j<prices.size();j++){
                if(prices[i]>=prices[j]){
                    prices[i]-=prices[j];
                    break;
                }
            }
        }
        return prices;
    }
};
```

## 复杂度分析

- 时间复杂度：`O(n^2)`
- 空间复杂度：`O(1)（不计算返回结果）`

## 关键细节与易错点

- 题目要求使用右侧第一个小于等于当前价格的商品作为折扣，因此找到后必须立即 break。
- 代码直接在 prices 上修改并返回，避免额外结果数组。
- 这份代码选择节省额外空间，额外空间为 O(1)，代价是最坏情况下需要 O(n^2) 时间。
- 如果使用单调栈，可以把时间优化到 O(n)，但需要 O(n) 额外空间保存还未找到折扣的下标。

## 可独立运行版本

### 输入输出说明

第一行输入整数 n，表示商品数量；第二行输入 n 个整数表示商品价格。程序输出折扣后的 n 个最终价格，数字之间以空格分隔。

### 示例输入

```text
5
8 4 6 2 3
```

### 示例输出

```text
4 2 4 2 3
```

### 完整可运行代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

class Solution {
public:
    vector<int> finalPrices(vector<int>& prices) {
        for (int i = 0; i < static_cast<int>(prices.size()); i++) {
            for (int j = i + 1; j < static_cast<int>(prices.size()); j++) {
                if (prices[i] >= prices[j]) {
                    prices[i] -= prices[j];
                    break;
                }
            }
        }
        return prices;
    }
};

int main() {
    int n;
    cin >> n;

    vector<int> prices(n);
    for (int& price : prices) {
        cin >> price;
    }

    Solution solution;
    vector<int> answer = solution.finalPrices(prices);
    for (int i = 0; i < static_cast<int>(answer.size()); i++) {
        if (i > 0) {
            cout << ' ';
        }
        cout << answer[i];
    }
    cout << '\n';
    return 0;
}
```

## 总结

直接模拟能清楚表达题意，也把额外空间压到 O(1)；如果更关注运行时间，可以用单调栈以 O(n) 空间换取 O(n) 时间。这里体现的是时间和空间之间的取舍。
