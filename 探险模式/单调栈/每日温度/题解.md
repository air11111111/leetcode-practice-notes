# LeetCode：每日温度

## 题目信息

- 题目链接：https://leetcode.cn/problems/daily-temperatures/description/?envType=problem-list-v2&envId=shujujiegouyusuanfa-xianxingqiantan-dandiaozhan
- 难度：中等
- 核心知识点：栈、数组、单调栈

## 题目概述

给定每天的温度数组，返回一个等长数组，其中每个位置表示需要等待多少天才能遇到更高温度；如果之后没有更高温度，则为 0。

## 解题思路

代码从右向左遍历温度数组，用栈保存右侧可能成为更高温度答案的下标。对于当前温度，栈中温度小于等于它的下标都不可能成为答案，因为当前这一天更靠左且温度不低，因此依次弹出。弹出后如果栈不为空，栈顶就是右侧最近的更高温度下标，二者差值就是等待天数；最后把当前下标压栈，供更左侧元素使用。

## 解题步骤

1. 初始化答案数组 v 和保存下标的栈。
2. 从右向左遍历 temperatures。
3. 弹出栈中所有温度小于等于当前温度的下标。
4. 若栈不为空，栈顶下标与当前下标的差值就是答案。
5. 将当前下标压入栈，继续处理更左侧的日期。

## 我的 LeetCode 代码

```cpp
class Solution {
public:
    vector<int> dailyTemperatures(vector<int>& temperatures) {
        int n = temperatures.size();
        vector<int> v(n, 0);
        stack<int> s;

        for (int i = n - 1; i >= 0; --i) {
            while (!s.empty() && temperatures[i] >= temperatures[s.top()]) {
                s.pop();
            }
            
            if (!s.empty()) {
                v[i] = s.top() - i;
            } else {
                v[i] = 0;
            }
            s.push(i);
        }
        return v;
    }
};
```

## 复杂度分析

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 关键细节与易错点

- 栈中保存的是下标，而不是温度值，这样可以计算等待天数。
- 从右往左处理时，栈顶始终是当前右侧最近且可能更高的温度位置。
- 遇到小于等于当前温度的下标要弹出，因为它们无法成为当前或更左侧较高温度的有效答案。
- 相同温度不算更高温度，因此条件中使用 >= 弹出相等温度。

## 可独立运行版本

### 输入输出说明

第一行输入整数 n，表示温度数组长度；第二行输入 n 个整数表示每天温度。程序输出 n 个整数，表示每一天等待更高温度所需天数。

```text

73 74 75 71 69 72 76 73
```

### 完整可运行代码

```cpp
#include <iostream>
#include <stack>
#include <vector>
using namespace std;

class Solution {
public:
    vector<int> dailyTemperatures(vector<int>& temperatures) {
        int n = temperatures.size();
        vector<int> v(n, 0);
        stack<int> s;

        for (int i = n - 1; i >= 0; --i) {
            while (!s.empty() && temperatures[i] >= temperatures[s.top()]) {
                s.pop();
            }

            if (!s.empty()) {
                v[i] = s.top() - i;
            } else {
                v[i] = 0;
            }
            s.push(i);
        }
        return v;
    }
};

int main() {
    int n;
    cin >> n;

    vector<int> temperatures(n);
    for (int& temperature : temperatures) {
        cin >> temperature;
    }

    Solution solution;
    vector<int> answer = solution.dailyTemperatures(temperatures);
    for (int i = 0; i < static_cast<int>(answer.size()); i++) {
        if (i > 0) {
            cout << ' ';
        }
        cout << answer[i];
    }
    cout << '\n';
    return 0;
}
```

## 总结

单调栈把“向右寻找下一个更高温度”的重复查找压缩成一次线性扫描，是这类下一个更大元素问题的典型做法。
