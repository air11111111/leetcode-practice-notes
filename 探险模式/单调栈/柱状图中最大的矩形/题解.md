# LeetCode：柱状图中最大的矩形

## 题目信息

- 题目链接：https://leetcode.cn/problems/largest-rectangle-in-histogram/?envType=problem-list-v2&envId=shujujiegouyusuanfa-xianxingqiantan-dandiaozhan
- 难度：困难
- 核心知识点：栈、数组、单调栈

## 题目概述

给定柱状图中每个柱子的高度，要求找出能够由连续柱子组成的最大矩形面积。

## 我最初的思路

我一开始尝试过暴力枚举矩形范围，但没有通过测试。后来查看题解后，改用单调栈维护柱子下标，才写出了这份代码。

## 解题思路

代码使用单调递增栈保存柱子的下标，并在栈底放入 -1 作为边界哨兵。遍历 heights 时，如果当前高度小于等于栈顶高度，说明栈顶柱子右边界已经确定，于是弹出栈顶并以该柱子高度计算矩形面积。弹出后新的栈顶就是左侧第一个更矮柱子的下标，当前 i 是右侧第一个不高于它的位置，因此宽度为 i-s.top()-1。遍历结束后，栈中剩余柱子的右边界统一视为数组末尾，再逐个结算面积。

## 解题步骤

1. 在栈中先压入 -1，作为左边界哨兵。
2. 从左到右遍历每个柱子下标。
3. 当当前高度小于等于栈顶柱子高度时，持续弹栈并计算被弹柱子的最大矩形面积。
4. 弹出柱子后，用当前下标作为右边界，用新栈顶作为左边界。
5. 遍历结束后，再处理栈中尚未结算的柱子，右边界视为 heights.size()。
6. 维护最大面积 res 并返回。

## 我的 LeetCode 代码

```cpp
class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        stack<int> s;
        s.push(-1);
        int res = 0;
        for(int i = 0; i < heights.size(); i++) {
            while(s.top() != -1 && heights[i] <= heights[s.top()]) {
                int height = heights[s.top()];
                s.pop();
                int width = i - s.top() - 1;
                res =  res > height * width ? res:height * width;
            }
            s.push(i);
        }
        
        while(s.top() != -1) {
            int height = heights[s.top()];
            s.pop();
            int width = heights.size() - s.top() - 1;
            res =  res > height * width ? res:height * width;
        }

        
        return res;
    } 
};
```

## 复杂度分析

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 关键细节与易错点

- 栈中保存的是下标，而不是高度，这样才能计算矩形宽度。
- 栈底的 -1 哨兵可以统一处理左边没有更矮柱子的情况。
- 弹出某个柱子时，它的高度已经找到了左右两侧第一个更矮边界。
- 使用 heights[i] <= heights[s.top()] 可以及时处理相等高度，避免重复宽度计算造成边界混乱。
- 暴力枚举范围通常会达到 O(n^2)，在大数据下容易超时；单调栈用空间换时间，将每个下标最多入栈、出栈一次。

### 输入输出说明

第一行输入整数 n，表示柱子数量；第二行输入 n 个整数表示每个柱子的高度。程序输出柱状图中最大矩形面积。

### 完整可运行代码

```cpp
#include <iostream>
#include <stack>
#include <vector>
using namespace std;

class Solution {
public:
    int largestRectangleArea(vector<int>& heights) {
        stack<int> s;
        s.push(-1);
        int res = 0;
        for (int i = 0; i < static_cast<int>(heights.size()); i++) {
            while (s.top() != -1 && heights[i] <= heights[s.top()]) {
                int height = heights[s.top()];
                s.pop();
                int width = i - s.top() - 1;
                res = res > height * width ? res : height * width;
            }
            s.push(i);
        }

        while (s.top() != -1) {
            int height = heights[s.top()];
            s.pop();
            int width = static_cast<int>(heights.size()) - s.top() - 1;
            res = res > height * width ? res : height * width;
        }

        return res;
    }
};

int main() {
    int n;
    cin >> n;

    vector<int> heights(n);
    for (int& height : heights) {
        cin >> height;
    }

    Solution solution;
    cout << solution.largestRectangleArea(heights) << '\n';
    return 0;
}
```

## 总结

这道题的难点在于快速找到每个柱子向左右能扩展到哪里。暴力做法直观但会超时；单调栈把边界查找压缩到一次线性扫描，是用 O(n) 空间换 O(n) 时间的典型优化。
