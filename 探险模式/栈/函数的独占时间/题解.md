# LeetCode：函数的独占时间

## 题目信息

- 题目链接：https://leetcode.cn/problems/exclusive-time-of-functions/description/?envType=problem-list-v2&envId=shujujiegouyusuanfa-xianxingqiantan-zhan
- 难度：中等
- 核心知识点：栈、数组

## 题目概述

给定若干函数调用日志，每条日志记录函数编号、开始或结束状态以及时间戳。函数可能嵌套调用，需要计算每个函数真正执行的独占时间。

## 解题思路

代码使用栈保存当前正在执行的函数编号，并用 last 记录上一段时间的起点。遇到 start 日志时，如果栈不为空，说明栈顶函数在 last 到当前 time 之前一直运行，因此先累加 time-last；然后把新函数压栈并更新 last。遇到 end 日志时，栈顶函数从 last 到 time 都在运行，结束时间戳也计入执行时间，所以累加 time-last+1，弹出该函数，并把 last 更新为 time+1。

## 解题步骤

1. 初始化答案数组、调用栈和 last 时间指针。
2. 逐条解析日志，得到函数 id、类型和时间戳。
3. 遇到 start 时，先结算当前栈顶函数到本次开始前的时间，再压入新函数。
4. 遇到 end 时，结算当前函数包含结束时间戳在内的执行时间，并弹出栈顶。
5. 遍历结束后返回每个函数的独占时间。

## 我的 LeetCode 代码

```cpp
class Solution {
public:
    vector<int> exclusiveTime(int n, vector<string>& logs) {
        vector<int> v(n, 0); 
        stack<int> s; 
        int last = 0;

        for (int i=0;i<logs.size();i++) {
            int pos1 = logs[i].find(':');
            int pos2 = logs[i].find(':', pos1 + 1);
            
            int id = stoi(logs[i].substr(0, pos1));
            string type = logs[i].substr(pos1 + 1, pos2 - pos1 - 1);
            int time = stoi(logs[i].substr(pos2 + 1));

            if (type == "start") {
                if (!s.empty()) {
                    v[s.top()] += time - last;
                }
                s.push(id);
                last = time;
            } else {
                v[s.top()] += time - last + 1;
                s.pop();
                last = time + 1;
            }
        }
        return v;
    }
};
```

## 复杂度分析

- 时间复杂度：`O(m)`
- 空间复杂度：`O(n)`

## 关键细节与易错点

- 调用栈的栈顶始终是当前正在运行的函数。
- end 日志的时间戳是闭区间，需要加 1。
- 函数结束后，下一段可计入时间从 time+1 开始。
- start 日志会暂停原栈顶函数，因此要先把原函数从 last 到 time 之前的时间累加。

## 可独立运行版本

### 输入输出说明

第一行输入两个整数 n 和 m，分别表示函数数量和日志数量；接下来输入 m 个形如 id:start:time 或 id:end:time 的日志字符串。程序输出 n 个整数，表示每个函数的独占时间。

### 示例输入

```text
2 4
0:start:0
1:start:2
1:end:5
0:end:6
```

### 示例输出

```text
3 4
```

### 完整可运行代码

```cpp
#include <iostream>
#include <stack>
#include <string>
#include <vector>
using namespace std;

class Solution {
public:
    vector<int> exclusiveTime(int n, vector<string>& logs) {
        vector<int> v(n, 0);
        stack<int> s;
        int last = 0;

        for (int i = 0; i < static_cast<int>(logs.size()); i++) {
            int pos1 = logs[i].find(':');
            int pos2 = logs[i].find(':', pos1 + 1);

            int id = stoi(logs[i].substr(0, pos1));
            string type = logs[i].substr(pos1 + 1, pos2 - pos1 - 1);
            int time = stoi(logs[i].substr(pos2 + 1));

            if (type == "start") {
                if (!s.empty()) {
                    v[s.top()] += time - last;
                }
                s.push(id);
                last = time;
            } else {
                v[s.top()] += time - last + 1;
                s.pop();
                last = time + 1;
            }
        }
        return v;
    }
};

int main() {
    int n, m;
    cin >> n >> m;

    vector<string> logs(m);
    for (string& log : logs) {
        cin >> log;
    }

    Solution solution;
    vector<int> answer = solution.exclusiveTime(n, logs);
    for (int i = 0; i < static_cast<int>(answer.size()); i++) {
        if (i > 0) {
            cout << ' ';
        }
        cout << answer[i];
    }
    cout << '\n';
    return 0;
}
```

## 总结

用栈维护嵌套调用关系，再用 last 切分连续时间段，可以清楚地区分当前运行函数和被暂停函数的执行时间。
