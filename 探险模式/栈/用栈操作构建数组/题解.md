# LeetCode：用栈操作构建数组

## 题目信息

- 题目链接：https://leetcode.cn/problems/build-an-array-with-stack-operations/description/?envType=problem-list-v2&envId=shujujiegouyusuanfa-xianxingqiantan-zhan
- 难度：中等
- 核心知识点：栈、数组、模拟

## 题目概述

给定一个严格递增的目标数组 target 和整数 n，只能按顺序读取 1 到 n，并使用 Push 与 Pop 操作构建目标数组，返回所需操作序列。

## 解题思路

代码使用 index 指向 target 中下一个需要构建的数字，并按顺序遍历 1 到 n。每读取一个数字都先执行 Push；如果当前数字等于 target[index]，就保留该数字并移动 index，否则立即执行 Pop 将其移除。当 target 已全部构建完成时提前结束。

## 解题步骤

1. 初始化 index，指向 target 的第一个元素。
2. 按顺序遍历数字 1 到 n，每次先记录一次 Push。
3. 若当前数字属于目标数组的下一个位置，则移动 index；否则记录一次 Pop。
4. 当 target 中所有元素都已构建完成时结束遍历并返回操作序列。

## 我的 LeetCode 代码

```cpp
class Solution {
public:
    vector<string> buildArray(vector<int>& target, int n) {
        vector<string> v;
        int index=0;
        for(int i=1;i<=n;i++){
            if(index>=target.size()){
                break;
            }
            v.push_back("Push");
            if(target[index]==i){
                index++;
            }else{
                v.push_back("Pop");
            }
        }
        return v;
    }
};
```

## 复杂度分析

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 关键细节与易错点

- 每读取一个数字都必须先执行 Push。
- 不需要的数字在 Push 后立即 Pop。
- target 构建完成后应提前结束，避免产生多余操作。

### 完整可运行代码

```cpp
#include <iostream>
#include <string>
#include <vector>
using namespace std;

class Solution {
public:
    vector<string> buildArray(vector<int>& target, int n) {
        vector<string> v;
        int index = 0;
        for (int i = 1; i <= n; i++) {
            if (index >= static_cast<int>(target.size())) {
                break;
            }
            v.push_back("Push");
            if (target[index] == i) {
                index++;
            } else {
                v.push_back("Pop");
            }
        }
        return v;
    }
};

int main() {
    int size, n;
    cin >> size >> n;

    vector<int> target(size);
    for (int& value : target) {
        cin >> value;
    }

    Solution solution;
    vector<string> operations = solution.buildArray(target, n);
    for (const string& operation : operations) {
        cout << operation << '\n';
    }
    return 0;
}
```

## 总结

顺序读取数字时，通过比较当前数字与 target 中下一个待构建元素，可以直接决定保留还是弹出，并在目标构建完成后及时停止。
