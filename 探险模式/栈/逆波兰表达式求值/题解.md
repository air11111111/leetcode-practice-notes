# LeetCode：逆波兰表达式求值

## 题目信息

- 题目链接：https://leetcode.cn/problems/evaluate-reverse-polish-notation/description/?envType=problem-list-v2&envId=shujujiegouyusuanfa-xianxingqiantan-zhan
- 难度：中等
- 核心知识点：栈、数组、数学

## 题目概述

给定一个表示逆波兰表达式的字符串数组，按照表达式顺序计算并返回整数结果。有效运算符包括加、减、乘、除，整数除法向零截断。

## 解题思路

代码使用栈保存尚未参与完整运算的数字。遍历 tokens 时，遇到数字就压栈；遇到运算符则依次弹出右操作数 num1 和左操作数 num2，计算 num2 运算 num1 后把结果重新压栈。遍历结束后，栈顶就是最终结果。

## 解题步骤

1. 创建一个整数栈。
2. 遍历每个 token，数字通过 stoi 转换后压栈。
3. 遇到运算符时，依次弹出右操作数和左操作数。
4. 按照运算符计算结果并重新压栈。
5. 遍历结束后返回栈顶结果。

## 我的 LeetCode 代码

```cpp
class Solution {
public:
    int evalRPN(vector<string>& tokens) {
        stack<int> s;
        for(int i=0;i<tokens.size();i++){
            if(tokens[i]=="+"||tokens[i]=="-"||tokens[i]=="*"||tokens[i]=="/"){
                int num1=s.top();
                s.pop();
                int num2=s.top();
                s.pop();
                if(tokens[i]=="+"){
                    s.push(num1+num2);
                }else if(tokens[i]=="-"){
                    s.push(num2-num1);
                }else if(tokens[i]=="*"){
                    s.push(num1*num2);
                }else{
                    s.push(int(num2/num1));
                }
            }else{
                s.push(stoi(tokens[i]));
            }
        }
        return s.top();
    }
};
```

## 复杂度分析

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 关键细节与易错点

- 第一次弹出的 num1 是右操作数，第二次弹出的 num2 是左操作数。
- 减法和除法必须计算 num2-num1 与 num2/num1，不能颠倒顺序。
- C++ 整数除法会向零截断，符合题目要求。

## 代码中需要注意的问题

- int(num2/num1) 的强制转换是冗余的，因为 num1 和 num2 都是 int，除法结果已经是整数。

## 可独立运行版本

### 输入输出说明

第一行输入整数 n，表示 token 数量；第二行输入 n 个以空格分隔的数字或运算符。程序输出逆波兰表达式的计算结果。

### 示例输入

```text

2 1 + 3 *
```

### 示例输出

```text

```

### 完整可运行代码

```cpp
#include <iostream>
#include <stack>
#include <string>
#include <vector>
using namespace std;

class Solution {
public:
    int evalRPN(vector<string>& tokens) {
        stack<int> s;
        for (int i = 0; i < static_cast<int>(tokens.size()); i++) {
            if (tokens[i] == "+" || tokens[i] == "-" || tokens[i] == "*" || tokens[i] == "/") {
                int num1 = s.top();
                s.pop();
                int num2 = s.top();
                s.pop();
                if (tokens[i] == "+") {
                    s.push(num1 + num2);
                } else if (tokens[i] == "-") {
                    s.push(num2 - num1);
                } else if (tokens[i] == "*") {
                    s.push(num1 * num2);
                } else {
                    s.push(int(num2 / num1));
                }
            } else {
                s.push(stoi(tokens[i]));
            }
        }
        return s.top();
    }
};

int main() {
    int n;
    cin >> n;

    vector<string> tokens(n);
    for (string& token : tokens) {
        cin >> token;
    }

    Solution solution;
    cout << solution.evalRPN(tokens) << '\n';
    return 0;
}
```

## 总结

逆波兰表达式天然适合使用栈处理；只要注意弹栈后的左右操作数顺序，就能在一次遍历中完成求值。
