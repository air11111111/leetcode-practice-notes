# LeetCode：有多少小于当前数字的数字

## 题目信息

- 题目链接：https://leetcode.cn/problems/how-many-numbers-are-smaller-than-the-current-number/description/?envType=problem-list-v2&envId=shujujiegouyusuanfa-xianxingqiantan-shuzu-ii
- 难度：简单
- 核心知识点：数组、哈希表、计数排序、排序

## 题目概述

给定一个整数数组，对数组中的每个元素，统计数组中严格小于它的元素数量，并按原数组顺序返回所有统计结果。

## 解题思路

代码先复制并排序数组。在有序数组中，一个值第一次出现的位置，恰好等于严格小于该值的元素数量。随后使用哈希表保存每个值出现的所有下标，再用第一个下标为原数组中的每个元素生成答案。

## 解题步骤

1. 复制原数组并对副本排序。
2. 遍历有序数组，将每个值对应的下标记录到哈希表中。
3. 遍历原数组，取对应值记录的第一个下标作为小于它的元素数量。
4. 返回结果数组。

## 我的 LeetCode 代码

```cpp
class Solution {
public:
    vector<int> smallerNumbersThanCurrent(vector<int>& nums) {
        vector<int> v=nums;
        sort(v.begin(),v.end());
        unordered_map<int,vector<int>> mp;
        for(int i=0;i<v.size();i++){
            mp[v[i]].push_back(i);
        }
        for(int i=0;i<nums.size();i++){
            v[i]=mp[nums[i]][0];
        }
        return v;
    }
};
```

## 复杂度分析

- 时间复杂度：`O(n log n)`
- 空间复杂度：`O(n)`

## 关键细节与易错点

- 排序后，一个值第一次出现的下标就是严格小于该值的元素个数。
- 必须使用第一次出现的位置，才能正确处理重复元素。

## 代码中需要注意的问题

- 哈希表中的 vector 实际只使用了第一个下标，可以简化为 unordered_map<int, int>，仅在每个值第一次出现时记录下标。

## 可独立运行版本

### 输入输出说明

第一行输入整数 n，表示数组长度；第二行输入 n 个整数。程序输出 n 个整数，分别表示小于每个原数组元素的数字数量。

### 示例输入

```text
5
8 1 2 2 3
```

### 示例输出

```text
4 0 1 1 3
```

### 完整可运行代码

```cpp
#include <algorithm>
#include <iostream>
#include <unordered_map>
#include <vector>
using namespace std;

class Solution {
public:
    vector<int> smallerNumbersThanCurrent(vector<int>& nums) {
        vector<int> v = nums;
        sort(v.begin(), v.end());
        unordered_map<int, vector<int>> mp;
        for (int i = 0; i < static_cast<int>(v.size()); i++) {
            mp[v[i]].push_back(i);
        }
        for (int i = 0; i < static_cast<int>(nums.size()); i++) {
            v[i] = mp[nums[i]][0];
        }
        return v;
    }
};

int main() {
    int n;
    cin >> n;

    vector<int> nums(n);
    for (int& num : nums) {
        cin >> num;
    }

    Solution solution;
    vector<int> answer = solution.smallerNumbersThanCurrent(nums);
    for (int i = 0; i < static_cast<int>(answer.size()); i++) {
        if (i > 0) {
            cout << ' ';
        }
        cout << answer[i];
    }
    cout << '\n';
    return 0;
}
```

## 总结

排序把大小关系转换为下标信息，再通过哈希表恢复原数组顺序，能够清晰地完成统计。
