# LeetCode：找到所有数组中消失的数字

## 题目信息

- 题目链接：https://leetcode.cn/problems/find-all-numbers-disappeared-in-an-array/description/?envType=problem-list-v2&envId=shujujiegouyusuanfa-xianxingqiantan-shuzu-ii
- 难度：简单
- 核心知识点：数组、哈希表

## 题目概述

给定一个长度为 n 的数组，其中每个元素都在 1 到 n 范围内，找出所有未在数组中出现的数字。

## 解题思路

代码尝试通过原地交换，让值 x 尽量移动到下标 x-1 的位置。第一轮完成位置整理后，第二轮扫描数组；若 nums[i] 不等于 i+1，则 i+1 就是没有出现的数字。为了避免重复值导致持续交换，代码使用 lastTemp 跳过刚处理过的重复值。

## 解题步骤

1. 遍历数组，已经位于正确位置的元素直接跳过。
2. 对错位元素，根据它的值将其交换到对应下标，并重新检查当前位置。
3. 遇到刚处理过的重复值时停止继续交换。
4. 再次遍历数组，把仍未放置正确的位置对应的数字加入答案。

## 我的 LeetCode 代码

```cpp
class Solution {
public:
    vector<int> findDisappearedNumbers(vector<int>& nums) {
        vector<int> v;
        int temp;
        int lastTemp;
        for(int i=0;i<nums.size();i++){
            if(nums[i]==i+1){
                continue;
            }
            if(lastTemp==nums[i]){
                continue;
            }
            lastTemp=nums[i];
            temp=nums[i];
            nums[i]=nums[nums[i]-1];
            nums[temp-1]=lastTemp;
            i--;
        }
        for(int i=0;i<nums.size();i++){
            if(nums[i]!=i+1){
                v.push_back(i+1);
            }
        }
        return v;
    }
};
```

## 复杂度分析

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)（不计算返回结果）`

## 关键细节与易错点

- 值 x 对应的正确位置是下标 x-1。
- 交换后通过 i-- 重新检查当前位置，直到当前位置正确或遇到重复值。
- 算法会修改输入数组。

## 代码中需要注意的问题

- 原始代码中的 lastTemp 未初始化便参与比较，属于 C++ 未定义行为，可能导致错误跳过元素。独立运行版将其初始化为 0；由于题目中的元素范围为 1 到 n，0 不会与数组元素冲突。

## 可独立运行版本

### 输入输出说明

第一行输入整数 n，表示数组长度；第二行输入 n 个范围在 1 到 n 内的整数。程序输出所有未出现的数字，数字之间以空格分隔。

### 示例输入

```text
8
4 3 2 7 8 2 3 1
```

### 示例输出

```text
5 6
```

### 完整可运行代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

class Solution {
public:
    vector<int> findDisappearedNumbers(vector<int>& nums) {
        vector<int> v;
        int temp;
        int lastTemp = 0;
        for (int i = 0; i < static_cast<int>(nums.size()); i++) {
            if (nums[i] == i + 1) {
                continue;
            }
            if (lastTemp == nums[i]) {
                continue;
            }
            lastTemp = nums[i];
            temp = nums[i];
            nums[i] = nums[nums[i] - 1];
            nums[temp - 1] = lastTemp;
            i--;
        }
        for (int i = 0; i < static_cast<int>(nums.size()); i++) {
            if (nums[i] != i + 1) {
                v.push_back(i + 1);
            }
        }
        return v;
    }
};

int main() {
    int n;
    cin >> n;

    vector<int> nums(n);
    for (int& num : nums) {
        cin >> num;
    }

    Solution solution;
    vector<int> answer = solution.findDisappearedNumbers(nums);
    for (int i = 0; i < static_cast<int>(answer.size()); i++) {
        if (i > 0) {
            cout << ' ';
        }
        cout << answer[i];
    }
    cout << '\n';
    return 0;
}
```

## 总结

利用值与目标下标之间的对应关系，可以在原数组上完成位置整理，并通过错位位置找出缺失数字；实现时必须确保用于判重的状态变量已经初始化。
