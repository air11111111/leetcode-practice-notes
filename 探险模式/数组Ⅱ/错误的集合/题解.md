# LeetCode：错误的集合

## 题目信息

- 题目链接：https://leetcode.cn/problems/set-mismatch/?envType=problem-list-v2&envId=shujujiegouyusuanfa-xianxingqiantan-shuzu-ii
- 难度：简单
- 核心知识点：数组、哈希表、位运算、排序

## 题目概述

原本包含 1 到 n 的集合因数据错误而出现一个重复数字，同时缺少一个数字。需要返回重复数字和缺失数字。

## 解题思路

代码先使用哈希表统计数组中每个数字出现的次数，再遍历 1 到 n：出现两次的数字就是重复值，出现零次的数字就是缺失值，最后按重复值、缺失值的顺序返回。

## 解题步骤

1. 遍历数组，使用哈希表记录每个数字的出现次数。
2. 遍历 1 到 n，找到出现次数为 2 的重复数字。
3. 在同一次遍历中找到出现次数为 0 的缺失数字。
4. 返回重复数字和缺失数字。

## 我的 LeetCode 代码

```cpp
class Solution {
public:
    vector<int> findErrorNums(vector<int>& nums) {
        unordered_map<int,int> mp;
        int x,y,flag=0;
        x=y=1;
        for(int i=0;i<nums.size();i++){
            mp[nums[i]]+=1;
        }
        for(int i=1;i<=nums.size();i++){
            if(mp[i]==2){
                x=i;
            }
            if(mp[i]==0){
                flag=1;
                y=i;
            }
        }
        if(flag==0){
            y=nums.size();
        }
        return {x,y};
    }
};
```

## 复杂度分析

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 关键细节与易错点

- 使用 mp[i] 查询不存在的键时，会自动插入该键并将值初始化为 0。
- 返回结果的顺序必须是重复数字在前、缺失数字在后。

## 代码中需要注意的问题

- flag 变量以及最后的 flag==0 分支是冗余的，因为第二次循环遍历了完整的 1 到 n，必然能够直接找到唯一缺失的数字。

## 可独立运行版本

### 输入输出说明

第一行输入整数 n，表示数组长度；第二行输入 n 个整数。程序输出两个整数，第一个是重复数字，第二个是缺失数字。

### 示例输入

```text
4
1 2 2 4
```

### 示例输出

```text
2 3
```

### 完整可运行代码

```cpp
#include <iostream>
#include <unordered_map>
#include <vector>
using namespace std;

class Solution {
public:
    vector<int> findErrorNums(vector<int>& nums) {
        unordered_map<int, int> mp;
        int x, y, flag = 0;
        x = y = 1;
        for (int i = 0; i < static_cast<int>(nums.size()); i++) {
            mp[nums[i]] += 1;
        }
        for (int i = 1; i <= static_cast<int>(nums.size()); i++) {
            if (mp[i] == 2) {
                x = i;
            }
            if (mp[i] == 0) {
                flag = 1;
                y = i;
            }
        }
        if (flag == 0) {
            y = nums.size();
        }
        return {x, y};
    }
};

int main() {
    int n;
    cin >> n;

    vector<int> nums(n);
    for (int& num : nums) {
        cin >> num;
    }

    Solution solution;
    vector<int> answer = solution.findErrorNums(nums);
    cout << answer[0] << ' ' << answer[1] << '\n';
    return 0;
}
```

## 总结

通过哈希表记录频次，可以直接根据出现次数识别重复数字和缺失数字，思路清晰且只需线性时间。
