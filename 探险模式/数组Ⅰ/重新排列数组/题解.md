# LeetCode：重新排列数组

## 题目信息

- 题目链接：https://leetcode.cn/problems/shuffle-the-array/description/?envType=problem-list-v2&envId=shujujiegouyusuanfa-xianxingqiantan-shuzu-i
- 难度：简单
- 核心知识点：数组

## 题目概述

给定一个由两部分组成的数组 nums，其中前半部分为 x1 到 xn，后半部分为 y1 到 yn，需要按 x1、y1、x2、y2 直至 xn、yn 的顺序重新排列。

## 解题思路

代码创建一个空结果数组，并遍历下标 0 到 n-1。每次先加入前半部分的 nums[i]，再加入后半部分对应的 nums[i+n]，从而按顺序交替合并两个长度为 n 的部分。

## 解题步骤

1. 创建空的结果数组。
2. 遍历前半部分的每个下标 i。
3. 依次将 nums[i] 和 nums[i+n] 加入结果数组。
4. 返回完成交替排列的结果数组。

## 我的 LeetCode 代码

```cpp
class Solution {
public:
    vector<int> shuffle(vector<int>& nums, int n) {
        vector<int> v;
        for(int i=0;i<n;i++){
            v.push_back(nums[i]);
            v.push_back(nums[i+n]);
        }
        return v;
    }
};
```

## 复杂度分析

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 关键细节与易错点

- 后半部分与前半部分对应元素的下标相差 n。
- 每轮循环向结果数组加入两个元素，最终恰好得到 2n 个元素。

## 可独立运行版本

### 输入输出说明

第一行输入整数 n；第二行输入 2n 个整数，前 n 个表示 x1 到 xn，后 n 个表示 y1 到 yn。程序输出重新排列后的 2n 个整数，整数之间以空格分隔。

### 示例输入

```text
3
2 5 1 3 4 7
```

### 示例输出

```text
2 3 5 4 1 7
```

### 完整可运行代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

class Solution {
public:
    vector<int> shuffle(vector<int>& nums, int n) {
        vector<int> v;
        for (int i = 0; i < n; i++) {
            v.push_back(nums[i]);
            v.push_back(nums[i + n]);
        }
        return v;
    }
};

int main() {
    int n;
    cin >> n;

    vector<int> nums(2 * n);
    for (int& num : nums) {
        cin >> num;
    }

    Solution solution;
    vector<int> answer = solution.shuffle(nums, n);
    for (int i = 0; i < static_cast<int>(answer.size()); i++) {
        if (i > 0) {
            cout << ' ';
        }
        cout << answer[i];
    }
    cout << '\n';
    return 0;
}
```

## 总结

利用两个部分对应位置之间固定的下标偏移 n，可以在一次遍历中完成交替排列。
