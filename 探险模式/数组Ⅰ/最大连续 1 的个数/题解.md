# LeetCode：最大连续 1 的个数

## 题目信息

- 题目链接：https://leetcode.cn/problems/max-consecutive-ones/?envType=problem-list-v2&envId=shujujiegouyusuanfa-xianxingqiantan-shuzu-i
- 难度：简单
- 核心知识点：数组

## 题目概述

给定一个只包含 0 和 1 的二进制数组，求数组中连续出现的 1 的最大数量。

## 解题思路

代码使用 cnt 记录当前连续 1 的长度，使用 count 记录遍历过程中出现过的最大长度。遇到 1 时增加 cnt 并更新最大值；遇到 0 时将 cnt 重置为 0。

## 解题步骤

1. 初始化当前连续长度 cnt 和最大连续长度 count。
2. 从左到右遍历数组。
3. 遇到 1 时增加 cnt，并用它更新 count。
4. 遇到 0 时将 cnt 清零。
5. 遍历结束后返回 count。

## 我的 LeetCode 代码

```cpp
class Solution {
public:
    int findMaxConsecutiveOnes(vector<int>& nums) {
        int n = nums.size();
        int count = 0;
        int cnt=0;
        for(int i=0;i<n;i++){
            if(nums[i]==1){
                cnt++;
                if(cnt>count){
                    count=cnt;
                }
            }else{
                cnt=0;
            }
        }
        return count;
    }
};
```

## 复杂度分析

- 时间复杂度：`O(n)`
- 空间复杂度：`O(1)`

## 关键细节与易错点

- 遇到 0 时必须重置当前连续长度。
- 每次遇到 1 后立即更新最大值，可以正确处理数组以连续 1 结尾的情况。

## 可独立运行版本

### 输入输出说明

第一行输入整数 n，表示数组长度；第二行输入 n 个只包含 0 和 1 的整数。程序输出数组中连续 1 的最大数量。

### 完整可运行代码

```cpp
#include <iostream>
#include <vector>
using namespace std;

class Solution {
public:
    int findMaxConsecutiveOnes(vector<int>& nums) {
        int n = nums.size();
        int count = 0;
        int cnt = 0;
        for (int i = 0; i < n; i++) {
            if (nums[i] == 1) {
                cnt++;
                if (cnt > count) {
                    count = cnt;
                }
            } else {
                cnt = 0;
            }
        }
        return count;
    }
};

int main() {
    int n;
    cin >> n;

    vector<int> nums(n);
    for (int& num : nums) {
        cin >> num;
    }

    Solution solution;
    cout << solution.findMaxConsecutiveOnes(nums) << '\n';
    return 0;
}
```

## 总结

使用一个变量记录当前连续段长度、另一个变量记录历史最大值，就能在一次遍历中得到答案。
