# LeetCode 刷题记录

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/air11111111/leetcode-practice-notes)

本仓库用于记录个人的 LeetCode 刷题代码、解题思路以及学习心得。

仓库内容纯属个人小白练习心得，题解和代码中难免存在错误或不足之处。如果你发现问题，或者有更加清晰、高效的解题方法，欢迎提出建议、指正与交流。

## 在线运行代码

点击上方 **Open in GitHub Codespaces** 按钮，在浏览器中创建在线 C++ 开发环境，无需在本地安装编译器。

环境启动后，在终端中运行指定题目的 `main.cpp`：

```bash
./scripts/run-cpp.sh "探险模式/数组Ⅱ/错误的集合"
```

脚本编译成功后，按照对应 `题解.md` 中的输入格式输入测试数据即可。

也可以直接传入源码路径：

```bash
./scripts/run-cpp.sh "探险模式/数组Ⅰ/最大连续 1 的个数/main.cpp"
```
